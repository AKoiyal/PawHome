# PawHome

PawHome is a community website project designed to improve user engagement through a multi-page structure and interactive front-end features.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Responsive design
- Sticky navigation
- Hero slideshow
- Animal filtering
- Modal popups
- Accordion sections
- Form validation

## Project Purpose
This project was developed to practice front-end development, website structure, and interactive user experience design.
