# PawHome

PawHome is a community-focused website project designed to support user engagement through a clear multi-page structure and interactive front-end features.

## Overview
This project was developed to practice front-end web development, responsive layout design, and user interaction design. The website presents a pet-related community platform with multiple pages and consistent styling across the site.

## Key Features
- Multi-page website structure
- Responsive design for different screen sizes
- Sticky navigation for better usability
- Hero slideshow on the home page
- Animal filtering feature
- Modal popup interactions
- Accordion sections for expandable content
- Form validation for user input

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
The project is organised into multiple pages, including:
- Home
- About
- Adopt
- Events
- Contact

## Purpose
The goal of this project was to strengthen practical skills in website structure, interface design, and interactive front-end development.
